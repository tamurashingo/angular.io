とてつもないアプリケーションを作りましょう

Angularは最新のWeb標準を使ってアプリケーションを構築するための開発環境です。Angularは次のような豊富な基本機能を持っています。ジェスチャー、アニメーション、フィルタ、ルーティング、データバインディング、セキュリティ、国際化、美しいコンポーネント。きわめてモジュール化されていて軽量で簡単に学ぶことができます。

とっとと始めましょう、とっとと作りましょう

クリーンで理解可能なコードとともにアイデアをぶちまけましょう。Angularはシンプルに作れて、簡単に変更できて、UXデザイナーとの協業も簡単です。 デフォルトで美しく、マテリアルデザインで、WebコンポーネントをサポートしているUIを作りましょう。クイックスタートガイドを使って数分以内に始めましょう。

@Component({ selector: 'my-app', template: '<h1>こんにちは {{ name }}</h1>' }) // Component controller class MyAppComponent { constructor() { this.name = 'Ali'; } }

Powered by the Open Source Community

Angularはあなたによるあなたのための製品です。私たちコミュニティはAngularの方向性を形作っています: コアとなる開発者は大企業だったり、中小企業だったり、学生、無所属のエキスパートです。私たちは成果物をオープンにしているため、Angularを改良したり、拡張したり変更したりすることができます。
